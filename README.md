# Infosys Stock Market Analysis Dashboard (2020–2025)

## Project Overview
This project analyzes Infosys stock performance from 2020 to 2025 using Python, Yahoo Finance, Pandas, and Tableau.

## Objectives
- Analyze stock price trends
- Calculate Daily Returns
- Compare MA20 and MA50 moving averages
- Analyze trading volume patterns
- Evaluate monthly stock performance
- Build an interactive Tableau dashboard

## Tools & Technologies
- Python
- Pandas
- NumPy
- yFinance
- Tableau Public

## Dashboard Features
### 📈 Stock Price Trend
Tracks Infosys closing price movement from 2020–2025.

### 📊 Trading Volume Analysis
Shows monthly trading activity and market participation.

### 📉 Daily Return Analysis
Highlights positive and negative return periods.

### 🔄 MA20 vs MA50 Analysis
Compares short-term and long-term moving averages.

### 📅 Monthly Performance Analysis
Displays monthly gains and losses using color-coded bars.
